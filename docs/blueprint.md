## Style Guidelines:

- Primary color: Warm terracotta (#E07A5F) to evoke a sense of home and traditional cooking.
- Background color: Soft beige (#F8F0E3) for a comforting and inviting atmosphere.
- Accent color: Olive green (#3D405B) to add a touch of sophistication and grounding.
- Body and headline font: 'Alegreya', a serif font to bring a human touch
- Clean and structured layout with clear sections for easy navigation.
- Use Lucide React icons to visually represent the main elements (food categories, cart, etc.).
